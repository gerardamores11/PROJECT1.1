About School Facility Feedback System

System Overview

The School Facility Feedback System is a simple web-based application designed to help students and staff report issues or provide feedback about school facilities. This system makes it easy to track and manage facility-related concerns efficiently.

Features

  • Easy-to-use feedback submission form
  • Multiple facility categories (Classroom, Restroom, Laboratory, Canteen, Library)
  • Star rating system (1-5 stars)
  • Admin dashboard to view all submitted feedback
  • Data persistence using browser localStorage

Technologies Used

  • HTML5 - Structure and content
  • CSS3 - Styling and layout
  • JavaScript - Interactivity and data management
  • LocalStorage API - Data persistence

Developer Information

How to Use

  1. Login with any username and password (demo mode)
  2. Navigate to the Feedback page to submit feedback
  3. Fill in the facility, description, and rating
  4. Click Submit to save your feedback
  5. View all submitted feedback in the Dashboard

Note: This is a demo system for educational purposes. All data is stored in your browser's localStorage and will be cleared if you clear your browser data.
